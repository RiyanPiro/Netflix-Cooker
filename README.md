# 🚀 Netflix Cooker
## 💥 Features
- No Proxies Required
- No Rate Limit
## ⚠️ Steps of Usage
1. Use convert.py to convert Json cookies in /convert/ folder into Netscape format (if not already) 
2. Paste .txt file cookies in /cookies/ folder of Netscape format
3. Run `pip install -r requirements.txt` command
4. Run main.py
